# FloProject — FHGR WINF1 Web Project

## What This Is

A complete web project for the FHGR Bachelor Sport Management module "Wirtschaftsinformatik: Grundlagen (WINF1)". The project delivers 4 graded artifacts (A-D) covering the full lifecycle of a sports website: topic definition, specification, wireframes, and a live website — all managed using HERMES methodology and presented cohesively in a GitHub repository.

## Core Value

Deliver a professional, cohesive set of deliverables that demonstrates mastery of the full web project lifecycle — from concept through implementation — with a creative sports topic that stands out.
